# Caso Practico
- En este video :
  
  1-  Actualizar el sistema
```
dnf update
```
  2-  Buscar el paquete **Nano**
```
dnf search nano
```
  
  3-  Instalar el paquete **Nano**
  
  ```
dnf install nano
```
  4-  Comprobar si el **Nano** ha instalado.
  ```
sudo dnf info nano
```
  
  5-  Desinstalar el paquete **NANO**
```
dnf remove nano
```
